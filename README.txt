使用方式：

1. 解壓縮本壓縮檔到你的 Hugo 專案根目錄（有 content/、themes/ 的那一層），允許覆蓋同名檔案。
2. 開啟 hugo.toml，把
   Your English Name / 你的中文名字
   改成你的真實名字。
3. 執行：
   hugo server -D --disableFastRender
4. 首頁會顯示滿版圖片＋ works/about/cv/contact 四個選單；上方語言切換按鈕已關閉。
5. 所有作品頁路徑為：/works/<slug>/，例如：/works/alkaline-pear-water/
6. 新作品「鹼性梨子水」已加入 2022 年度。

作品說明目前是佔位文字，請自行到 content/works/<slug>/index.md 中編輯中英內容。
